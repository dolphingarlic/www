Overview

This unit contains TWinPEImageResourceReader, a TAbstractResourceReader descendant that is able to read Microsoft Windows PE image files (executables, dynamic link libraries and so on).

Adding this unit to a program's uses clause registers class TWinPEImageResourceReader with TResources.
