TVersionFixedInfo.FileOS

The operating system the file was designed to run on

Declaration

Source position: versiontypes.pp line 66

public property TVersionFixedInfo.FileOS : LongWord
  read fFileOS
  write fFileOS;

Description

It is one of the following values:

  • VOS_UNKNOWN
  • VOS_DOS
  • VOS_OS216
  • VOS_OS232
  • VOS_NT

combined with one of the following values:

  • VOS__WINDOWS16
  • VOS__WINDOWS32
  • VOS__PM16
  • VOS__PM32

Note: some predefined combinations do exist:

  • VOS_DOS_WINDOWS16
  • VOS_DOS_WINDOWS32
  • VOS_OS216_PM16
  • VOS_OS232_PM32
  • VOS_NT_WINDOWS32
