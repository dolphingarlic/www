4.2.7 The descriptions node

The descriptions tag below the package tag contains one description tag for each description file belonging to the package. No other tags should appear below the descriptions tag.

Example:

<descriptions>  
  <description file="rtl.xml"/>  
  <description file="system.xml"/>  
</descriptions>
