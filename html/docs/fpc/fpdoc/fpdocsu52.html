4.2.4 The package node

Each package tag under the packages tag describes one package. It contains at most 2 tags: the units tag and the descriptions tag.

The package tag has 3 attributes:

name
This attribute specifies the name of the package. This attribute is mandatory.
output
This attribute contains the default output name for the package. This attribute is optional.
content
The content attribute specifies the name of the content file to write for this package. This attribute is optional. No file will be written if it is omitted.

Example:

<package name="rtl" output="rtl" content="rtl.xct">
