1.3.10 $FRAMEWORKPATH : Specify framework path.

This option serves to specify the framework search path on Darwin, where the compiler looks for framework files. Used as

{$FRAMEWORKPATH XXX}

it will add XXX to the framework path. The value XXX can contain one or more paths, separated by semicolons or colons.
