1.2.7 $CALLING : Specify calling convention

The {$CALLING } directive tells the compiler which calling convention should be used if none is specified:

{$CALLING REGISTER}

By default it is REGISTER. The following calling conventions exist:

default
register
cdecl
pascal
safecall
stdcall
oldfpccall

For a more detailed explanation of calling conventions, see section 6.3, page 396. As a special case, DEFAULT can be used, to restore the default calling convention.
