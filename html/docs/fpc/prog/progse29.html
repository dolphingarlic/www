7.2 Making libraries

Free Pascal supports making shared or static libraries in a straightforward and easy manner. If you want to make static libraries for other Free Pascal programmers, you just need to provide a command line switch. To make shared libraries, refer to the chapter 12, page 633. If you want C programmers to be able to use your code as well, you will need to adapt your code a little. This process is described first.
