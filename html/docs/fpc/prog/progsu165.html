8.2.11 Record types

Each field of a record is stored in a contiguous sequence of variables, where the first field is stored at the lowest address in memory. In case of variant fields in a record, each variant starts at the same address in memory. Fields of a record are usually aligned, unless the packed directive is specified when declaring the record type.

For more information on field alignment, consult section 8.3.2, page 536.
