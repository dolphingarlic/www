6.4.11 The Help menu

The ”Help” menu provides entry points to all the help functionality of the IDE, as well as the means to customize the help system.

Contents
Shows the help table of contents
Index
(SHIFT-F1) Jumps to the help Index.
Topic search
(CTRL-F1) Jumps to the topic associated with the currently highlighted text.
Previous topic
(ALT-F1) Jumps to the previously visited topic.
Using help
Displays help on using the help system.
Files
Allows the configuration of the help menu. With this menu item, help files can be added to the help system.
About
Displays information about the IDE. See section 6.13.3, page 366 for more information.
