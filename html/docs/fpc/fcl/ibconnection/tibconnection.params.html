published property TIBConnection.Params : TStrings;

Description

Params is a #rtl.classes.TStringList of name=value combinations that set database-specific parameters.

The following parameter is supported:

  • PAGE_SIZE: size of database pages (an integer), e.g. 16384.

See your database documentation for more details.
