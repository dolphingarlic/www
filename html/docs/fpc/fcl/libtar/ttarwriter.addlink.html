Arguments

Filename

Linkname

  

Target of hard link

DateGmt

  

Timestamp of entry

Description

AddLink adds a hard link entry to the archive. The entry has name FileName, timestamp DateGmt and points to LinkName.

The link entry is written with the current owner data and permissions.
