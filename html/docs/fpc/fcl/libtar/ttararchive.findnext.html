TTarArchive.FindNext

Find next archive entry

  var DirRec: TTarDirRec

):Boolean;

Arguments

DirRec

  

Information about the archive

Function result

True if the next entry was found.

Description

FindNext positions the file pointer on the next archive entry, and returns all information about the entry in DirRec. It returns True if the operation was succeful, or False if not (for instance, when the end of the archive was reached).

Errors

In case there are no more entries, False is returned.
