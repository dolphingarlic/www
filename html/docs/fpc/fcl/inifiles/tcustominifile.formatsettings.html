TCustomIniFile.FormatSettings

Format settings to use when reading date/time/float values

Declaration

Source position: inifiles.pp line 159

public TCustomIniFile.FormatSettings : TFormatSettings;

Description

FormatSetting is a set of formatting settings similar to Sysutils.FormatSettings. If TCustomIniFile.FormatSettingsActive is True, then the value of this field will be used to read values in TCustomIniFile.ReadDateTime, TCustomIniFile.ReadDate, TCustomIniFile.ReadTime and TCustomIniFile.ReadFloat or when writing values in TCustomIniFile.WriteDateTime, TCustomIniFile.WrireDate, TCustomIniFile.WriteTime and TCustomIniFile.WriteFloat.
