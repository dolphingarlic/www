TJSONArray.Strings

Get or set elements as string values

Declaration

Source position: fpjson.pp line 452

public property TJSONArray.Strings[Index: Integer] : TJSONStringType
  read GetStrings
  write SetStrings;

Description

Strings gives direct access to the TJSONData.AsString property when reading. Reading it is the equivalent to accessing 

Items[Index].AsString

When writing, it will check if a string JSON value is located at the given location, and replace it with the new value. If a non-string value is there, it is replaced with the written string value.
