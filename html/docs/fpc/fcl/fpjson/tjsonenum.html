String Key value

Numerical key value

  Value: TJSONData;

  

Actual value

end;

Description

TJSONEnum is the loop variable type to use when implementing a JSON enumerator (for in). It contains 3 elements which are available in the loop: key, keynum (numerical key) and the actual value (TJSONData).
