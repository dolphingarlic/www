function Application: TCustomDaemonApplication;

Function result

TCustomDaemonApplication instance used by application

Description

Application is the TCustomDaemonApplication instance used by this application. The instance is created at the first invocation of this function, so it is possible to use RegisterDaemonApplicationClass to register an alternative TCustomDaemonApplication class to run the application.
