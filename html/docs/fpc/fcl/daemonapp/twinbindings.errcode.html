Description

ErrCode contains a service specific error code that is reported with TDaemon.ReportStatus to the windows service manager. If it is zero, then the contents of Win32ErrCode are reported. If it is nonzero, then the windows-errorcode is set to ERROR_SERVICE_SPECIFIC_ERROR.
