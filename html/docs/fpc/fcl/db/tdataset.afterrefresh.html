Event triggered after the data has been refreshed.

public property TDataSet.AfterRefresh : TDataSetNotifyEvent

Description

AfterRefresh is triggered at the end of the TDataset.Refresh method, after the dataset has refreshed its data and is again in dsBrowse state. This event can be used to react on changes in data in the current record
