TParam.SetData

Set the parameter value from a buffer

Arguments

Buffer

  

Buffer to read data from

Description

SetData performs the rever operation of TParam.GetData: it reads the parameter value from the memory area pointed to by Buffer. The size of the data read is determined by TParam.GetDataSize and the type of data by TParam.DataType : it is the same storage mechanism used by TField, and so can be used to copy the value from a TDataset record buffer.

Errors

Not all field types are supported. If an unsupported field type is encountered, an EDatabaseError exception is raised.
