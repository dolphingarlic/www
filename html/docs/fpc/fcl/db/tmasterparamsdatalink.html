Description

TMasterParamsDataLink is a TDataLink descendent that can be used to establish a master-detail relationship between 2 TDataset instances where the detail dataset is parametrized using a TParams instance. It takes care of closing and opening the detail dataset and copying the parameter values from the master dataset whenever the data in the master dataset changes.
