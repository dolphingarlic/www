TProcess.WindowTop

Y-coordinate of the initial window (Windows only)

Declaration

Source position: process.pp line 159

published property TProcess.WindowTop : Cardinal
  read dwY
  write SetWindowTop;

Description

WindowTop is the initial Y coordinate (in pixels) of the process' main window, relative to the top border of the desktop. This property is only effective if suoUsePosition is specified in StartupOptions
