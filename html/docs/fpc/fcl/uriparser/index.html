Overview

The URIParser unit contains a basic type (TURI) and some routines for the parsing (ParseURI) and construction (EncodeURI) of Uniform Resource Indicators, commonly referred to as URL: Uniform Resource Location. It is used in various other units, and in itself contains no classes. It supports all protocols, username/password/port specification, query parameters and bookmarks etc..
