Description

TIPCClientComm is an abstract component which implements the client-side communication protocol. The behaviour expected of this class must be implemented in a platform-dependent descendent class.

The TSimpleIPCClient class does not implement the messaging protocol by itself. Instead, it creates an instance of a (platform dependent) descendent of TIPCClientComm which handles the internals of the commnication protocol.

The server side of the messaging protocol is handled by the TIPCServerComm component. The descenent components must always be implemented in pairs.
