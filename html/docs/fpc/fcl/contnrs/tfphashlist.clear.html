TFPHashList.Clear

Clear the list

Declaration

Source position: contnrs.pp line 242

public procedure TFPHashList.Clear;

Description

Clear removes all items from the list. It does not free the data items themselves. It frees all memory needed to contain the items.

Errors

None.
