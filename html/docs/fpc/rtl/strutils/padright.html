PadRight

Add spaces to the right of a string till a certain length is reached.

Declaration

Source position: strutils.pp line 165

  const S: string;

  N: Integer

):string;

Arguments

S

  

String to pad.

N

  

Minimal length of the resulting string.

Function result

String of length N, padded to the right with spaces

Description

PadRight add spaces to the left of the string S till the result reaches length Len. If the string S has length equal to or largert than Len, no spaces are added, and the string S is returned as-is. The resulting string is S, left-justified on length Len.

Errors

None.
