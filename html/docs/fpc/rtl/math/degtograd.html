degtograd

Convert degree angle to grads angle

Declaration

Source position: math.pp line 279

function degtograd(

  deg: Float

):Float;

Description

Degtograd transforms it's argument deg (an angle in degrees) to grads. (90 degrees is 100 grad.)

Errors

None.
