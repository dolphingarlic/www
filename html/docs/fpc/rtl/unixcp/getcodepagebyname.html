GetCodepageByName

Find code page by name

):TSystemCodePage;

Arguments

cpname

  

Name of code page.

Function result

Code page number corresponding to cpname or CP_NONE if none found.

Description

GetCodepageByName returns the code page number matching cpname. The supported code page names are the ones commonly in use in libiconv. Names are searched case-sentively, with the exception that 'cpN' is converted to 'CPN', where N is a digit.

Errors

If no matching code page name is found, CP_NONE is returned.
