MinuteOfTheMonth

Calculate number of minutes elapsed since the start of the month.

Arguments

AValue

  

Moment in the month.

Function result

Number of minutes elapsed since the start of the month.

Description

MinuteOfTheMonth returns the number of minutes that have passed since the start of the Month (00:00:00) till the moment indicated by AValue. This is a zero-based number, i.e. 00:00:59 on the first day of the month will return 0.

For an example, see the WeekOfTheMonth function.
