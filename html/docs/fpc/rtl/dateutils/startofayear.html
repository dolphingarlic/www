StartOfAYear

Return the first day of a given year.

Declaration

Arguments

AYear

  

Year to get first date from.

Function result

First day of AYear as a TDateTime

Description

StartOfAYear returns a TDateTime value with the date of the first day of the year AYear (January 1).
