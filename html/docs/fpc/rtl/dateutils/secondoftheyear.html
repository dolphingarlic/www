SecondOfTheYear

Calculate the number of seconds elapsed since the start of the year.

Arguments

AValue

  

Moment in the year

Function result

Number of seconds elapsed since the start of the year.

Description

SecondOfTheYear returns the number of seconds that have passed since the start of the year (January 1, 00:00:00) till the moment indicated by AValue. This is a zero-based number, i.e. January 1 00:00:00.999 will return 0.

For an example, see the WeekOfTheYear function.
