Description

EncodeDateTime encodes the values AYearAMonth, ADay,AHour, AMinute,ASecond and AMilliSecond to a date/time valueand returns this value.

For an example, see DecodeDateTime.

Errors

If any of the arguments is not valid, then an EConvertError exception is raised.
