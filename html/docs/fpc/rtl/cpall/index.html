Overview

The cpall unit registers all known single-byte codepages: 1251, 866, ISO 8856-5 (cyrillic), 8859-1, 8859-2, 1253 (greek), 850, 437, 1252, 646, 874, 856, 1250, 1254, 1255, 1256, 1257, 1258, 852.

This unit does not contain any routines. It simply uses the other units so all corresponding code pages are registered.
