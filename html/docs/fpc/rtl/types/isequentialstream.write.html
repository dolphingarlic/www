  pv: Pointer;

  cb: DWORD;

  pcbWritten: PDWord

):HRESULT;

Description

Write writes cbCount bytes from the memory pointed to by pv to the stream and returns the number of bytes written in pcbwritten. The result is zero for success or an error code.
