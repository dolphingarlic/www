Dseg

Return data segment

Declaration

Source position: systemh.inc line 1063

function Dseg: Word;

Description

DSeg returns the data segment register. In Free Pascal, it returns always a zero, since Free Pascal is a 32/64 bit compiler.

Errors

None.
