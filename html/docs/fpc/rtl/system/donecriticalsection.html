DoneCriticalsection

Clean up a critical section.

Declaration

Source position: threadh.inc line 161

procedure DoneCriticalsection(

  var cs: TRTLCriticalSection

);

Description

DoneCriticalsection cleans up the critical section CS. After a call to DoneCriticalsection, the critical section can no longer be used with EnterCriticalsection or LeaveCriticalsection, unless it is again initialized with InitCriticalSection
