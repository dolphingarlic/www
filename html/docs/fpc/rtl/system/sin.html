sin

Calculate sine of angle

Declaration

Source position: mathh.inc line 95

function sin(

  d: ValReal

):ValReal;

Description

Sin returns the sine of its argument X, where X is an angle in radians. If the absolute value of the argument is larger than 2^63, then the result is undefined.

Errors

None.
