DefaultTextLineBreakStyle

Default line ending style.

Declaration

Source position: sysunixh.inc line 56

const DefaultTextLineBreakStyle: TTextLineBreakStyle = tlbsLF;

Description

DefaultTextLineBreakStyle contains the default OS setting for the TTextLineBreakStyle type. It is initizliaed by the system unit, and is used to determine the default line ending when writing to text files.

This constant is part of a set of constants that describe the OS characteristics. These constants should be used instead of hardcoding OS characteristics.
