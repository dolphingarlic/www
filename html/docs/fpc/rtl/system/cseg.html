Cseg

Return code segment

Declaration

Source position: systemh.inc line 1062

function Cseg: Word;

Description

CSeg returns the Code segment register. In Free Pascal, it returns always a zero, since Free Pascal is a 32/64 bit compiler.

Errors

None.
