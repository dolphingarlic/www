The system unit

Overview

The wincrt unit provides some auxiliary routines for use with the graph unit, namely keyboard support. It has no connection with the crt unit, nor with the Turbo-Pascal for Windows WinCrt unit. As such, it should not be used by end users. Refer to the crt unit instead.
