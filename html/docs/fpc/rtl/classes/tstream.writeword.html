TStream.WriteWord

Write a word to the stream.

Declaration

Source position: classesh.inc line 890

public procedure TStream.WriteWord(

  w: Word

);

Description

WriteWord writes the word W (i.e. 2 bytes) to the stream. This is a utility function which simply calls the Write function. The word can be read from the stream using the ReadWord function.

Errors

If an error occurs when attempting to write, an EStreamError exception will be raised.
