TStringStream.Read

Reads from the stream.

  var Buffer;

  Count: LongInt

):LongInt; override;

Description

Read overrides the Read from TStream. It tries to read Count bytes into Buffer. It returns the number of bytes actually read. The position of the stream is advanced with the number of bytes actually read; When the reading has reached the end of the DataString, then the reading stops, i.e. it is not possible to read beyond the end of the datastring.
