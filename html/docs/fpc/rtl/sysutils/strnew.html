function strnew(

  p: PWideChar

):PWideChar; overload;

Description

Copies P to the Heap, and returns a pointer to the copy.

Errors

Returns Nil if no memory was available for the copy.

Const P1 : PChar = 'This is a PChar string';

  P2:=StrNew (P1);
  If P1=P2 then
    writeln ('This can''t be happening...')
  else
    writeln ('P2 : ',P2);
  StrDispose(P2);  
end.
