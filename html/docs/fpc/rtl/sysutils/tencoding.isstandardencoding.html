TEncoding.IsStandardEncoding

Check if the encoding is one of the standard encodings

Declaration

Source position: sysencodingh.inc line 58

public class function TEncoding.IsStandardEncoding(

Description

IsStandardEncoding will return True is one of the standard encoding instances TEncoding.ANSI, TEncoding.ASCII, TEncoding.Default, TEncoding.BigEndianUnicode, TEncoding.Unicode, TEncoding.UTF7 or TEncoding.UTF8. Otherwise it returns False.
