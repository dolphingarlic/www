LeftStr

Return a number of characters starting at the left of a string.

Declaration

Source position: sysstrh.inc line 263

function LeftStr(

  const S: string;

  Count: Integer

):string;

Description

LeftStr returns the Count leftmost characters of S. It is equivalent to a call to Copy(S,1,Count).

Errors

None.
