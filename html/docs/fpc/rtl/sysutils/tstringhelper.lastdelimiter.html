TStringHelper.LastDelimiter

Return the last position of one of a series of delimiters

Declaration

Source position: syshelph.inc line 112

public function TStringHelper.LastDelimiter(

  const Delims: string

):Integer;

Description

LastDelimiter searches the string backwards for an occurence of one of the characters in Delims, and returns the corresponding (zero-based) index in the string. If no occurrence is found, -1 is returned.
