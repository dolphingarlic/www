strlicomp

):SizeInt;

Description

Compares maximum L characters of the null-terminated strings S1 and S2, ignoring case. The result is

  • A negative Longint when S1<S2.
  • 0 when S1=S2.
  • A positive Longint when S1>S2.

For an example, see StrIComp
