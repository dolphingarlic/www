AppendStr

Append one ansistring to another.

Declaration

Source position: sysstrh.inc line 70

procedure AppendStr(

  var Dest: string;

  const S: string

);

Description

AppendStr appends S to Dest.

This function is provided for Delphi compatibility only, since it is completely equivalent to Dest:=Dest+S.

Errors

None.
