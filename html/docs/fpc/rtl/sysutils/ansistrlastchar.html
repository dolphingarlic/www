AnsiStrLastChar

Return a pointer to the last character of a string.

Declaration

Source position: sysstrh.inc line 96

function AnsiStrLastChar(

  Str: PChar

):PChar;

Description

Return a pointer to the last character of the null-terminated string.

Remark: A widestring manager must be installed in order for this function to work correctly with various character sets. If none is installed, this function is the same as @S[Length[S]].

Errors

None.
