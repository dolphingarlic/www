AnsiCompareFileName

Compare 2 filenames.

  const S1: string;

):SizeInt;

Description

AnsiCompareFileName compares 2 filenames S1 and S2, and returns

< 0
if S1<S2.
= 0
if S1=S2.
> 0
if S1>S2.

The function actually checks FileNameCaseSensitive and returns the result of AnsiCompareStr or AnsiCompareText depending on whether FileNameCaseSensitive is True or False

Errors

None.
