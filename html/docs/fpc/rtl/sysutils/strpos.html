strpos

Find position of one null-terminated substring in another.

function strpos(

  str1: PWideChar;

  str2: PWideChar

):PWideChar;

Description

Returns a pointer to the first occurrence of S2 in S1. If S2 does not occur in S1, returns Nil.

Errors

None.
