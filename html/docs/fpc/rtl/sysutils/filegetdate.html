FileGetDate

Return the file time of an opened file.

Declaration

Source position: filutilh.inc line 176

function FileGetDate(

  Handle: THandle

):LongInt;

Description

FileGetdate returns the filetime of the opened file with filehandle Handle. It is the same as FileAge, with this difference that FileAge only needs the file name, while FilegetDate needs an open file handle.

Errors

On error, -1 is returned.

See also

FileAge

  

Return the timestamp of a file.

Example

Program Example39;

{ This program demonstrates the FileGetDate function }

Uses sysutils;

Var F,D : Longint;

Begin
  F:=FileCreate('test.dat');
  D:=FileGetDate(F);
  Writeln ('File created on ',DateTimeToStr(FileDateToDateTime(D)));
  FileClose(F);
  DeleteFile('test.dat');
End.
