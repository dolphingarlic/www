Description

ToString will, in the class method version, convert the AValue WordBool to a string representation. In the function method version the WordBool value itself (Self) will be converted.

If the UseBoolStrs parameter equals TUseBoolStrs.True, then the string representation will use the WordBool strings BoolStrs. The default value for UseBoolStrs is TUseBoolStrs.False.

The conversion is done using the BoolTostr function.
