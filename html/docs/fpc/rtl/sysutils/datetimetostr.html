Description

DateTimeToStr returns a string representation of DateTime using the formatting specified in LongDateTimeFormat. It corresponds to a call to FormatDateTime('c',DateTime) (see formatchars).

Note that if the time part is 0 (i.e. midnight), no time is appended.

Errors

None.
