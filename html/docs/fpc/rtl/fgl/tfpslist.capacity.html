TFPSList.Capacity

Current capacity of the list

public property TFPSList.Capacity : Integer

Description

Capacity is the current capacity (maximum amount of elements) of the list. The list capacity will expand automatically if an item is added and the capacity is reached (i.e. TFPSList.Count equals capacity. Expanding the list is an expensive operation involving reallocation of memory and moving of list data in memort, so capacity can be set to a large amount to avoid frequent reallocations.
